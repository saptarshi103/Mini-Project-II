🏠 Room Rental Platform

A full-stack web application for landlords to list rooms and tenants to browse, filter, and get directions to rooms. Built with Node.js, MySQL, and AWS, focusing on secure backend functionality, cloud integration, and a seamless user experience.



✨ Features
Landlord

✅ Register and login securely

✅ Post, update, and delete room listings

✅ Upload multiple images per room

✅ Manage all listings via a dashboard

Tenant / User

🔎 Browse available rooms with details and images

📞 Contact landlords directly

🗺️ View room location on map and get directions

🏙️ Filter rooms by city, price, or type


Authentication

🔐 Sign in via Email, Google, or Facebook

🔑 JWT token-based secure login

🧑‍💼 Role-based access for landlords


Cloud & Storage

☁️ Image storage using AWS S3

🌐 Optionally deploy backend on AWS EC2


🛠 Tech Stack

Backend: Node.js, Express.js

Database: MySQL, Sequelize ORM

Frontend: HTML, CSS, JavaScript

Authentication: JWT, OAuth (Google & Facebook) (Implemented via AWS Cognito)

Cloud: AWS S3, AWS Cognito, (EC2, RDS, Route 53 – optional for hosting)

Maps & Geolocation: Map integration using Leaflet
